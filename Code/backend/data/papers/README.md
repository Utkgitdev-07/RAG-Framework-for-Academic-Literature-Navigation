# Papers Directory

Place your PDF academic papers in this directory.

## Instructions

1. Add PDF files directly here
2. Files should have `.pdf` extension
3. Recommended: 5-20 papers for testing
4. The system will automatically process them when you click "Index Papers"

## Supported Formats

- PDF files (.pdf)
- Maximum recommended size: 50MB per file

## Example Structure

```
papers/
├── paper1.pdf
├── paper2.pdf
├── paper3.pdf
└── ...
```

## Notes

- Papers are processed automatically during indexing
- Metadata is extracted from PDFs
- Text is extracted for embedding
- No manual configuration needed
